Fruit Merge

A Physics-Based Puzzle Game

View Project

Overview

Fruit Merge is a casual physics-based puzzle game where players drop fruit into a container, merge matching fruit to create larger ones, and manage risk as the pile grows.

This project was created as a small but complete solo game, with a focus on clean mechanics, readable feedback, and WebGL browser performance.
