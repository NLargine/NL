Overview

A browser-based incremental RPG built from scratch using vanilla JavaScript. The project focuses on scalable combat systems, modular architecture, mission-based progression, and clean UI structure — all deployed via GitHub Pages.

Play Live Demo

My Role: Solo Developer

This project was built as a systems-focused portfolio piece to demonstrate scalable game architecture, modular UI rendering, and progression design in a browser environment without frameworks.

  • Game Systems Architecture (Combat, Scaling, Progression)
  • Mission & Multi-Wave Encounter Design
  • Itemization & Rarity System
  • Skill Unlock & Stat Modifier Framework
  • Modular UI Rendering (Separation of Logic & Presentation)
  • State Management & Data Structuring
  • Frontend UI/UX Design (Vanilla JS + CSS)
  • Deployment via GitHub Pages

Core Systems & Architecture

Combat Engine

Modular combat system with crit chance, evasion, scaling enemies, and status effects. Enemy stats dynamically scale based on player progression.

Mission Framework

Multi-wave missions with area modifiers that globally affect combat, creating variation and strategic decision-making.

Item & Rarity System

Items use a rarity-based multiplier system with dynamic stat calculation rather than hardcoded values.

Scaling & Progression

XP curve, unlockable skills, and structured progression tiers designed to demonstrate incremental game design principles.

Modular UI Rendering

UI separated from core game logic using dedicated render functions to maintain maintainability and scalability.

Vanilla Implementation

Built without frameworks to showcase understanding of core JavaScript, DOM manipulation, and state management.
